Программа структурирования файлов.
По умолчанию - переименовывает по шаблону [год]_[№ месяца]_[№ файла]
С ключом -r - рекурсивное извлечение из всех подпапок и переименование (несовместим с [-f])
С ключом -f - распределяет файлы по подпапкам с именем [год]_[№ месяца]
Год и номер месяца - берутся из даты съемки если это фото, иначе из даты изменения файла.
Путь к корневой директории указать в аргументах программы.
